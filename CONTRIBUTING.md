# Contributing

Thank you for your interest in NetScan.

At this stage the project is still stabilizing and **pull requests are
not actively accepted**.

However, feedback and suggestions are welcome.

## Ways to Contribute

You can still help improve NetScan by:

-   Reporting bugs
-   Suggesting new features
-   Submitting device fingerprints
-   Improving documentation

Please open a GitHub Issue describing your suggestion.

## Code Style

NetScan follows simple Python guidelines:

-   Python 3.10+
-   Clear and readable code
-   Minimal external dependencies

## Future Contributions

Once the project stabilizes, contribution guidelines and pull request
workflows will be enabled.
